# Responsive-Restaurant-Website-
A modern and responsive restaurant website developed using HTML5, CSS3, JavaScript, Bootstrap, and SCSS.
This project is designed to provide an engaging and visually appealing user experience through interactive UI components, responsive layouts, smooth hover effects, and well-structured web pages.

The website includes multiple sections such as Home, About, Services, Menu, Booking, Team, Testimonials, and Contact, making it suitable for showcasing a complete restaurant business interface.

_Features_
1.Fully responsive design for all screen sizes
2.Interactive hover effects and smooth user interface
3.Modern navigation bar and homepage layout
4.Restaurant menu and service showcase
5.Online table booking section
6.Team and customer testimonial pages
7.Clean and organized frontend structure
8.Bootstrap-based responsive components
9.SCSS support for better styling customization

_Technologies Used_
HTML5
CSS3
JavaScript
Bootstrap
SCSS

_Project Structure_
index.html – Homepage
about.html – About section
service.html – Services page
menu.html – Food menu section
booking.html – Table booking page
team.html – Team information
testimonial.html – Customer testimonials
contact.html – Contact section
Purpose of the Project

This project was developed to enhance frontend web development skills and demonstrate practical implementation of responsive web design, Bootstrap components, interactive animations, and modern UI/UX concepts.
